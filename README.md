# arduino-line-follower-basic
 
Kit robot: https://ardushop.ro/ro/home/658-kit-robot-line-follower.html și instrucțiuni de asamblare.

 1. Instalare Arduino IDE<br />
    Accesați acest link și descărcați/instalați ultima versiune pentru Arduino IDE: https://www.arduino.cc/en/software
 2. L298N Library<br />
    Pentru controlul motoarelor avem nevoie de următoarea librărie: https://github.com/AndreaLombardo/L298N/
    Se descarcă conform instrucțiunilor și se instalează în Arduino IDE.
 3. Instalare program arduino-line-fallower-basic<br />
    Se descarcă conținutul acestui repository (Click pe butonul verde Code -> Download ZIP), se dezarhivează local și se deschide fișierul 
    src\sketch_dec30a\sketch_dec30a.ino din Arduino IDE.
 4. Setarea mediului de lucru    
    Accesează meniul 'Tools' din Arduino IDE și setezi:<br />
    Board: "Arduino Nano"<br />
    Processor: "ATmega328P (Old Bootloader)"<br />
